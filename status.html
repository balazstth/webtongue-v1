Working Features

* Core: let, var, if, eq, gt, gteq, and, or, not, loop, run, exit, error, debug, ws, tab (completed)

* Basic mathematics: inc, dec, add, sub, mul, div, mod, pow, sin, cos, tan, asin, acos, atan, ln, e, pi, abs, ceil, floor, round, rand (completed)

* Stack and Queue: push, pop, lift, drop, peek, poke, top, dump, tostack, tostring, melt, merge, get (completed)

* Miscellany and basic I/O: date, print, println, input (completed)

* Mezzanine subroutines: nop, string, size, read, write, ins, del, sqrt, exp, log, torad, todeg, dup, swp, rot, clr (completed)

Known Flaws

none
